# 🚀 Contact Manager CLI (Python)

A powerful and user-friendly command-line application to manage contacts efficiently using Python.

---

## 📌 Features

- Add new contacts  
- View all contacts with total count  
- Edit existing contacts  
- Delete contacts  
- Search contacts by name / phone / email  
- Mark and view favorite contacts ⭐  
- Prevent duplicate entries  
- Export contacts to CSV  
- Data stored using JSON  

---

## 🛠️ Technologies Used

- Python  
- JSON  
- CSV  

---

## ▶️ How to Run

```bash
python main.py
```
